## 2020-09-15

Дедлайн: 2020-09-22

### 5

- вводится положительное целое число `n`
- вывести YES, если `n` представимо в виде суммы квадратов двух неотрицательных целых

```
25
YES
```

потому что `25 = 9 + 16`

```
10
YES
```

потому что `10 = 1 + 9`

```
7
NO
```

потому что `7 = 1 + 6 = 2 + 5 = 3 + 4`
